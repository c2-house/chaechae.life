**2024년 5월 27일부터 ~ 2024년 6월 16일까지** 진행된다고 하니 참여해보면 좋을 것 같습니다.

## 경품

- TDD의 저자로 유명한 켄트 벡의 신간(**켄트 벡의 Tidy First 1권**)이 경품이라 욕심이 좀 납니다.
- 커피없이 못사는 저에겐 스벅커피 쿠폰도 나쁘지 않은 것 같습니다.
- 참여만 해도 마일리지를 준다니 안할 그냥 참여하는게 이득인 것 같습니다.

<Image src="/hanbit-coding-knowhow-event/01.jpg" alt="한빛 개발자 이벤트 경품" />

## 나의 팁 공유

다들 알고 있는 것들일테니 팁이라하기도 민망하네요.

- 남의 코드 많이 보면서 배우기 - 관심있는 분야의 오픈소스 코드를 자주 구경하는 편입니다.
- GPT이후로 AI를 잘 활용하기 위해 프롬프트 많이 작성해보고 공부를 꾸준히 하고 있습니다.
- 개발하기 전에 해결해야하는 문제에 대한 이해를 최대한 먼저하고 진행하려고 합니다.

## 마무리

저는 참여했습니다. 이렇게 책이 경품으로 걸린 이벤트 너무 좋은 것 같습니다. 당첨되고 싶네요.
